# 📱 Personal Flutter Project – First Client App

This is my first personal mobile application project built using Flutter. It serves as a real-world practice app focused on building a simple, clean, and user-friendly experience.

## 📖 About the Project

This application is designed as a simple and accessible transaction logging system with a focus on clean UI and usability. The goal is to make it easy for users to view and manage transaction records without complexity.

## ❤️ Personal Note

This is my first ever client-based project, and my first client is my mom. I built this application specifically to help her manage and view transaction records in a simple and organized way.

## 🛠️ Tech Stack

* 🐦 **Flutter**
* 🎯 **Dart**
* 💾 **Local Database** (Isar)
* ⚙️ **State Management** (Provider)

## ✨ Features

* 📝 Transaction logging system
* 🌗 Light/Dark theme support
* 🔠 Adjustable font scaling for accessibility
* 🎨 Simple navigation and clean UI
* 📊 Export and backup features

## 🎯 Purpose of This Project

This project represents my first step in creating real-world mobile applications for actual users, focusing on usability, clarity, and practical features.
